# Aura-Calendar
This project is a Discord bot created for the AURA Pok√©mon League server. The bot is designed to allow the creation of events on a Google Calendar.

# Features
- Create events in Google Calendar from Discord
- List upcoming events
- Link to the calendar

# Command List
- `event` : Create a new event
- `see-event` : See the last twenty events
- `calendar` : Get a link to the calendar
